         /**
     * 将十六进制的字符串转换成字节数组
     *
     * @param hexString
     * @return
     */
    public static byte[] hexStrToByteArrs(String hexString) {
        if (StringUtils.isEmpty(hexString)) {
            return null;
        }
 
        hexString = hexString.replaceAll(" ", "");
        int len = hexString.length();
        int index = 0;
 
        byte[] bytes = new byte[len / 2];
 
        while (index < len) {
            String sub = hexString.substring(index, index + 2);
            bytes[index / 2] = (byte) Integer.parseInt(sub, 16);
            index += 2;
        }
 
        return bytes;
    }
 
    /**
     * 数组转换成十六进制字符串
     *
     * @param byte[]
     * @return HexString
     */
    public static final String bytesToHexString(byte[] bArray) {
        StringBuffer sb = new StringBuffer(bArray.length);
        String sTemp;
        for (int i = 0; i < bArray.length; i++) {
            sTemp = Integer.toHexString(0xFF & bArray[i]);
            if (sTemp.length() < 2)
                sb.append(0);
            sb.append(sTemp.toUpperCase());
        }
        return sb.toString();
    }

1. 十六进制转byte

   ```
   int n=Integer.parseInt("C0",16);
   byte b=(byte) n;
   System.out.println(n);	//192
   System.out.println(b);	//-64
   ```

   int占4个字节，byte占一个字节，一个字节8位，强制类型转换int时截取低8位，对数据不会影响。

   如果byte转成int，会产生一个-128-127的有符号字节，所以byte转int时需要判断符号

   ```
   int n=Integer.parseInt("C0",16);
   byte b=(byte) n;
   b = b > 0 ? b : b + 256;
   System.out.println(n);	//192
   System.out.println(b);	//192
   
   ```

2. byte转十六进制字符串
   ```
   byte b=-64;
   int b=b>0?b:b+256;
   String s=Integer.toString(b,16);
   System.out.println(s);	//C0
   ```
